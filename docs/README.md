# Documentación de Unidata Portal

Esta carpeta contiene la documentación técnica y funcional de los módulos que integran el portal **Unidata**.

## Estructura de la Carpeta
- `log_cambios_semanal.md`: Registro de los avances realizados en los módulos de Artículos, Homologación, DB Master, Descargas y Seguridad.

## Cómo Contribuir
Al finalizar una funcionalidad importante o completar una jornada de cambios significativos, se debe actualizar el log semanal o crear un archivo específico si la funcionalidad es muy extensa.
