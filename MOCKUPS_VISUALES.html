Mockups Visuales - Unidata

1. Dashboard de Sincronización

Vista Principal: Centraliza el estado de todas las sucursales conectadas.

  • Estado en Tiempo Real: Indicadores visuales de conexión para cada base de datos.
  • Notificaciones de Sync: Interfaz moderna para el monitoreo de tareas en segundo plano.
  • Sidebar Colapsable: Navegación eficiente entre los diferentes módulos técnicos.

2. Módulo de Carga y Mapeo

Carga masiva: Interfaz diseñada para la gestión de miles de artículos.

  • Carga fluida de archivos CSV/Excel.
  • Mapeo inteligente de columnas a campos de base de datos.
  • Sistema de "Revertir" para seguridad de datos masivos.
